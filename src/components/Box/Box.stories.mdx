# Box

The box component allows you to create layouts for any component.

## Flex box

Create a flex layout using props.

  <Story story={FlexBox} />

## Gird box

Create a grid layout using the **cols** and **span** props.

<Source
  language='tsx'
  code={dedent`
<Box cols> {/* automatically divided in 12 columns */}
  <Box span={3}></Box>
  <Box span={9} cols> {/* automatically divided in 9 columns */}
    <Box span={6}></Box>
    <Box span={3}></Box>
  </Box>
</Box>
  `}
/>
